# Docker swarm 이란?
Docker swarm 은 도커를 위한 클러스터링 툴이다.<br/>
하나 또는 그 이상의 도커 호스트들을 이용해서 Swarm 클러스터를 구성할 수 있으며 Kubernetes 클러스터 매니저처럼 Swarm 또한 Swarm 클러스터 내의 유효한 자원을 기반으로 컨테이너들을 관리할 수 있다.<br/>
